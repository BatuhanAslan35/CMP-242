# CMP-242
Beginner-OOP2-Example
This project is a simple Python Object-Oriented Programming (OOP) example that demonstrates the use of classes, inheritance, class methods, and basic polymorphism through a student GPA tracking system. It is a beginner-level project and does not include professional or advanced programming methods.

The project includes two classes: Student and StudentStatus. The Student class handles basic student information such as name and GPA, keeps track of the total number of students, and calculates the average GPA. The StudentStatus class inherits from Student and adds functionality to determine whether a student has passed or failed based on their GPA.

Features
	•	Create multiple students and store their GPA data
	•	Count total number of students using a class method
	•	Calculate average GPA across all students
	•	Determine pass or fail status based on GPA
	•	Demonstrates simple inheritance and polymorphism


File Structure:
student.py          → Base class (Student)
studentstatus.py    → Derived class (StudentStatus)
main.py             → Main script demonstrating functionality
